STEP 2 · DATA
--------------------------------------------------------------------------------

  Choose "Upload files" and select file-3.csv from this folder
  (participant/scripts/seed-data/file-3.csv).

  The portal auto-detects the 16 columns and previews the first 5 rows.
  Detected column types (verify in the preview):
    time       int      (Unix timestamp, e.g. 1636467153)
    ram_limit  str      (e.g. "75M")
    cpu_limit  float    (e.g. 0.5)
    ram_usage  float    (e.g. 43921408.0)
    cpu_usage  float    (e.g. 0.064585)
    n          int      (requests, e.g. 100)
    c          int      (concurrency, e.g. 1)
    lat50 … lat100  int (latency percentiles in microseconds)
